アローラ!!/ポーズ(初回生産限定盤)にモンコレGETが付属

April 12, 2017

CD 初回限定盤

アニメ「ポケットモンスター サン&ムーン」の主題歌CDの初回限定盤にモンコレGETが特典として付いて発売されました。

発売日

今日、4月12日発売です。
定価は、￥1,599(税込)ですが、すでに各オンラインストアでは売り切れ、Amazonでは価格が高騰しています。

内容

「ピカチュウ バトルポーズ(Zワザver.)」

今までピカチュウは以下の種類が発売されていました。

  • モンコレGET vol.4のノーマルver.
  • モンコレGET vol.4のZワザver.
  • ボイスいっぱい！モンスターボール、付属のピカチュウ バトルポーズ

今回は、モンスターボール付属のバトルポーズピカチュウのZワザver.となるようです。

感想

完全にチェック漏れでした。
気がついた時にはすでに遅し。各所完売済み。
しかし、ピカチュウはすでに３種類持っているため、高騰した価格での購入も悩みどころ。

vol.4のZワザバージョンではZワザは電気タイプのスパーキングギガボルト、でしたが、今回のバトルポーズのZワザは何になるのでしょうか。
ピカチュウ専用Zワザ、「ひっさつのピカチュ〜ト」が出るのでしょうか。出るのならちょっと欲しいですが。
(もしくは、映画前売り券に付属するサトピカZのワザ、「1000まんボルト」は…さすがに出ないですかね）
